# Proje Görselleri ve Grafikler

Bu klasör, akademik makale ve sunum için R ve Python kullanılarak üretilmiş yüksek çözünürlüklü (300 DPI) grafikleri içerir.

### Figür Listesi:

* **`Figure1_Heatmap.png`:** Tüm hasta kohortunun metabolik heterojenliğini gösteren ısı haritası.
* **`Figure2_Pathways.png`:** Kanser türleri arasındaki metabolik yolak farklarını gösteren kutu grafikleri.
* **`Figure3_SERS_Spectrum.png`:** *In silico* simüle edilmiş SERS spektral imzaları.
* **`Figure4_ROC_Curves.png`:** Tanısal modelin performans eğrileri (AUC değerleri).
* **`Figure5_Survival.png`:** Nükleotit skoruna dayalı Kaplan-Meier sağkalım analizi.
* **`Figure6_Oncogenes.png`:** Metabolik değişimleri yöneten ana sürücü genlerin (KRAS, TP53 vb.) ifadesi.
